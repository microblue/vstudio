# 🚀 剧本创作功能 - 快速体验指南

**体验时间**: 3分钟  
**前置条件**: 本地开发环境已启动

---

## 📱 体验步骤

### 1️⃣ 访问应用 (10秒)
🌐 **打开浏览器**: http://localhost:5173

### 2️⃣ 演示登录 (5秒)  
✨ **点击**: "使用演示账号登录" 按钮  
自动填充并登录演示账号

### 3️⃣ 进入剧本创作 (5秒)
📝 **左侧菜单**: 点击 "剧本创作" ✨  
进入AI创作界面

### 4️⃣ 设置创作参数 (30秒)
📋 **填写故事大纲** (示例):
```
一个内向的程序员在咖啡厅遇到了一位艺术家，两人从最初的陌生到逐渐熟悉，在城市的各个角落展开了一段温馨的爱情故事。强调真实自然的情感交流，避免过于戏剧化的情节。
```

🎭 **选择参数**:
- 剧本类型: 爱情剧
- 目标时长: 5-8分钟  
- AI模型: Claude Sonnet (推荐)

### 5️⃣ 开始生成 (60秒)
🚀 **点击**: "开始生成剧本"  
📺 **观看**: 流式打字机效果  
⏱️ **等待**: 约60秒完成生成

### 6️⃣ 保存并编辑 (30秒)
💾 **点击**: "保存并继续"  
📝 **进入**: 剧本编辑器  
✏️ **体验**: 编辑模式切换

### 7️⃣ 查看历史版本 (20秒)  
📜 **右侧面板**: "AI创作历史"  
🔄 **点击**: "加载" 按钮测试版本切换
